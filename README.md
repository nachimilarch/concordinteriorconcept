# Concorde Interior Concepts — Website & CMS

## Tech Stack
- Frontend : React + Vite + Tailwind CSS + Framer Motion
- Backend  : Node.js + Express.js
- Database : MySQL
- Auth     : JWT
- Upload   : Multer (local) / AWS S3

## Quick Start

### 1. Database
```sql
mysql -u root -p < server/models/schema.sql
```

### 2. Backend
```bash
cd server
npm install
npm run dev
```

### 3. Frontend
```bash
cd client
npm install
npm run dev
```
